Connect your WordPress store to the gapShop high-performance C# backend.

== Description ==
gapShop provides enterprise-grade infrastructure for small merchants. 
* **Fintech:** Integrated Stripe Connect for usage-based billing.
* **Shipping:** One-click labels via Shippo.
* **AI SEO:** Automated product descriptions via Cloudinary.

== Installation ==
1. Upload the `gapshop.zip` file via the WordPress 'Plugins' menu.
2. Activate the plugin.
3. Click the "gapShop" menu item to connect your account.
